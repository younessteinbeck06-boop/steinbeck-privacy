# Datenschutzerklärung – Steinbeck

Die App „Steinbeck“ dient der Verwaltung von Inventar und Produkten.

## 1. Datenspeicherung
Alle eingegebenen Daten werden entweder lokal auf dem Gerät gespeichert oder – bei optionaler Anmeldung – zur Synchronisation zwischen mehreren Geräten gespeichert.

## 2. Benutzerkonto
Die Erstellung eines Benutzerkontos ist freiwillig.
Das Konto dient ausschließlich dazu, Daten zwischen mehreren Geräten zu synchronisieren und Mitarbeiterzugriffe zu verwalten.

## 3. Erhobene Daten
Es werden ausschließlich folgende Daten gespeichert:
- E-Mail-Adresse (bei Registrierung)
- Produktdaten (Name, Menge, optional Bilder)
- Einstellungen innerhalb der App

## 4. Weitergabe von Daten
Es erfolgt keine Weitergabe personenbezogener Daten an Dritte.

## 5. Werbung & Tracking
Die App enthält keine Werbung und verwendet keine Tracking-Dienste.

## 6. Löschung
Nutzer können ihre Daten jederzeit in der App löschen.
